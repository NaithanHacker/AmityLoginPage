A frontend clone of the official Amizone Login Portal built using pure HTML5 and CSS3. This project recreates the visual structure and design style of the Amizone authentication page while focusing on responsive layout, clean UI design, and frontend development practice.

📌 About The Project

This project is a recreation of the Amizone login interface made for learning and practice purposes. The goal was to improve frontend development skills by rebuilding a real-world university portal UI using only HTML and CSS without any frameworks or libraries.

The project demonstrates:

UI cloning techniques
Layout structuring
Responsive design
Styling modern login forms
GitHub Pages deployment
